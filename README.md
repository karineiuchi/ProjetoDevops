# ProjetoDevops

## 📌 Descrição
Projeto Gerenciador de Tarefas

## 🛠 Tecnologias Utilizadas

### Backend
- Node.js – ambiente de execução JavaScript
- Express 5 – framework para criar a API
- SQLite3 – banco de dados
- CORS – permite requisições entre diferentes origens
- Swagger UI Express – documentação da API

### Ferramentas de desenvolvimento
- Nodemon – reinicia automaticamente o servidor durante o desenvolvimento

### Frontend
- HTML

## ⚡ Executar

```bash
git clone https://github.com/karineiuchi/ProjetoDevops.git
cd ProjetoDevops/backend
npm install
npm run dev
```

# Depois abra o arquivo index.html no navegador.
